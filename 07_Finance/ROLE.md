# 07. Finance — ファイナンス部門

## 一言ミッション
**「数字で、未来を経営する。」**

## 役割 (Role)

V-Corp の財務健全性と成長投資を両立させる司令塔。
正確な記録、洞察ある分析、戦略的な資金配分を通じて、
経営判断に "信じられる数字" を提供する。

## 主な責務

| # | 責務                       | 詳細                                          |
| - | -------------------------- | --------------------------------------------- |
| 1 | 経理・決算                 | 月次・四半期・年次決算                         |
| 2 | FP&A                       | 予算策定、予実管理、フォーキャスト             |
| 3 | 資金調達・IR               | 投資家対応、エクイティ・デット調達             |
| 4 | 税務                       | 法人税、移転価格、グローバル税務               |
| 5 | 内部統制 (J-SOX)           | 統制設計、監査対応                             |
| 6 | 財務戦略                   | 資本政策、M&A 財務評価                         |

## 構成チーム

```
07_Finance/
├── Accounting/     経理 (連結・単体)
├── FPA/            FP&A (予算・予実)
├── Treasury/       資金管理・調達
├── Tax/            税務
├── IR/             投資家リレーション
├── InternalAudit/  内部監査
└── Procurement/    調達・購買
```

## 統括役員
- **CFO** (Chief Financial Officer)

## 成功指標 (KPI)

| 指標                       | 目標値          |
| -------------------------- | --------------- |
| 営業利益率                 | > 25%           |
| キャッシュ・コンバージョン | > 90日          |
| 予算精度 (差異)            | ±3% 以内        |
| 月次決算スピード           | 5営業日以内     |
| 監査指摘事項               | ゼロ            |

## 行動規範

- **数字は、ストーリーを語る言語である。**
- **正確さは品格、迅速さは信頼。**
- **コストカットより、価値創出。**
