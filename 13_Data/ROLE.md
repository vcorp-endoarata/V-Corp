# 13. Data — データ部門

## 一言ミッション
**「データを、意思決定の燃料に変える。」**

## 役割 (Role)

V-Corp 全社のデータを収集・整備・分析・活用する部門。
単なる "数字を出す" 部隊ではなく、ビジネスに洞察と予測を届ける戦略パートナー。
すべての意思決定がデータドリブンであるための基盤を築く。

## 主な責務

| # | 責務                       | 詳細                                            |
| - | -------------------------- | ----------------------------------------------- |
| 1 | データ基盤構築             | データレイク、DWH、ETL/ELT                       |
| 2 | BI / ダッシュボード        | 全社 KPI 可視化、セルフサービス BI               |
| 3 | データサイエンス           | 予測モデル、需要予測、チャーン予測               |
| 4 | 機械学習エンジニアリング   | ML パイプライン、MLOps                            |
| 5 | アナリティクス             | 事業分析、A/B テスト分析                          |
| 6 | データガバナンス           | データ品質、メタデータ、プライバシー              |
| 7 | データリテラシー教育       | 全社員のデータ活用力向上                          |

## 構成チーム

## 統括役員
- **CTO 管掌 / Chief Data Officer (CDO)**

## 成功指標 (KPI)

| 指標                          | 目標値             |
| ----------------------------- | ------------------ |
| データパイプライン稼働率      | > 99.9%            |
| ダッシュボード月間アクセス数  | 全社員の 80% 以上  |
| ML モデル本番投入数 / 年       | 30本以上           |
| データ品質スコア              | > 95%              |
| データ要求 → 提供 リードタイム | < 3営業日          |
