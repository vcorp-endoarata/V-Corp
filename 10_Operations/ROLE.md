# 10. Operations — オペレーション部門

## 一言ミッション
**「会社という機械を、最も滑らかに動かす。」**

## 役割 (Role)

V-Corp の事業が日々止まらず、効率よく回り続けるための裏方部門。
プロセス設計、業務自動化、コスト最適化を通じて、
他部門が価値創出に集中できる土台を築く。

## 主な責務

| # | 責務                       | 詳細                                          |
| - | -------------------------- | --------------------------------------------- |
| 1 | ビジネスオペレーション     | 部門横断プロセス設計・改善                     |
| 2 | プロジェクトマネジメント   | 全社プログラム・PMO                            |
| 3 | 業務自動化                 | RPA、ワークフロー自動化                        |
| 4 | サプライチェーン           | 物流・在庫管理（ハードウェア事業）             |
| 5 | ファシリティ               | オフィス・リモートワーク環境                   |
| 6 | 戦略実行支援               | OKR 運用、戦略マイルストーン管理               |
| 7 | ベンダー管理               | 外部委託先のマネジメント                       |

## 構成チーム

```
10_Operations/
├── BizOps/             ビジネスオペレーション
├── PMO/                プロジェクト・プログラム管理
├── Automation/         業務自動化・RPA
├── Facilities/         オフィス・ファシリティ
├── Procurement/        ベンダー・購買管理
├── SupplyChain/        サプライチェーン
└── Strategy/           戦略実行・OKR運用
```

## 統括役員
- **COO** (Chief Operating Officer)

## 成功指標 (KPI)

| 指標                       | 目標値           |
| -------------------------- | ---------------- |
| 業務自動化による工数削減   | 年 20% 削減      |
| プロセスエラー率           | < 0.1%           |
| OKR 達成率                 | > 70%            |
| 全社員満足度 (環境)        | > 4.3 / 5.0      |
| ベンダー SLA 遵守率        | > 99%            |

## 行動規範

- **目立たぬ仕事こそ、最大の貢献。**
- **ボトルネックを見つけ、外し続けよ。**
- **ルールはユーザー (社員) のためにある。**
