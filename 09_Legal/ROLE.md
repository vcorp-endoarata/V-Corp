# 09. Legal — リーガル部門

## 一言ミッション
**「事業の翼に、ブレーキではなく地図を。」**

## 役割 (Role)

V-Corp が法令・倫理・契約を遵守しながら、最大のスピードで事業展開できるよう支援する部門。
"No と言う" のではなく、"How に変える" 戦略法務を実践する。

## 主な責務

| # | 責務                       | 詳細                                            |
| - | -------------------------- | ----------------------------------------------- |
| 1 | 契約レビュー・交渉         | 顧客・パートナー契約の審査                       |
| 2 | コーポレートガバナンス     | 取締役会、株主総会、議事録                       |
| 3 | 知的財産                   | 特許出願、商標管理、ライセンス                   |
| 4 | コンプライアンス           | 各国規制、業界規制への対応                       |
| 5 | プライバシー / GDPR        | データ保護、PIA                                  |
| 6 | 紛争・訴訟対応             | 訴訟、調停、和解交渉                             |
| 7 | 倫理・サステナビリティ     | 倫理ポリシー、ESG 開示                            |

## 構成チーム

```
09_Legal/
├── Corporate/      会社法・ガバナンス
├── Commercial/     商事契約
├── IP/             知的財産
├── Privacy/        プライバシー・データ保護
├── Compliance/     コンプライアンス
├── Litigation/     紛争・訴訟
└── Ethics/         倫理・サステナビリティ
```

## 統括役員
- **CHRO 管掌 / 法務担当 GC** (General Counsel)

## 成功指標 (KPI)

| 指標                       | 目標値              |
| -------------------------- | ------------------- |
| 契約レビュー Turnaround    | < 3営業日           |
| コンプライアンス違反件数   | ゼロ                |
| 訴訟敗訴率                 | < 5%                |
| 特許出願件数               | 年 50件以上         |
| 全社員コンプラ研修受講率   | 100%                |

## 行動規範

- **守るのではなく、護りながら攻める。**
- **法律は最低ライン、倫理は私たちの誇り。**
- **シンプルな契約は、信頼の証。**
