# 08. People — ピープル部門

## 一言ミッション
**「人が、すべて。」**

## 役割 (Role)

V-Corp の最大の資産である "人" を採用し、育て、活かし、繋ぎ止める部門。
従来の "人事" を超え、組織文化・働き方・成長機会を設計する戦略パートナー。

## 主な責務

| # | 責務                      | 詳細                                            |
| - | ------------------------- | ----------------------------------------------- |
| 1 | タレントアクイジション    | 採用戦略・採用ブランディング                     |
| 2 | オンボーディング          | 入社後30/60/90日の成功支援                       |
| 3 | L&D (Learning & Dev)      | 研修・キャリア開発・1on1文化                     |
| 4 | カルチャー                | バリューの体現、エンゲージメント                 |
| 5 | DEI (多様性・公平・包摂)  | 多様な人材が活躍できる環境                       |
| 6 | コンペンセーション        | 報酬設計、エクイティ                             |
| 7 | パフォーマンス            | 評価制度、フィードバック文化                     |

## 統括役員
- **CHRO** (Chief Human Resources Officer)

## 成功指標 (KPI)

| 指標                          | 目標値             |
| ----------------------------- | ------------------ |
| 従業員 NPS (eNPS)             | > 70               |
| Voluntary Turnover            | < 8%               |
| 採用充足率                    | > 95%              |
| 内定承諾率                    | > 85%              |
| 内部昇格比率                  | > 60%              |
| 女性管理職比率                | > 40%              |

## 行動規範

- **人は資源ではない。共創者だ。**
- **採用は最重要業務。リーダーの仕事の20%。**
- **退職は別れではなく、卒業である。**
