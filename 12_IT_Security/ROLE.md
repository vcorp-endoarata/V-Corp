# 12. IT & Security — IT・セキュリティ部門

## 一言ミッション
**「全社員の生産性と、全顧客の信頼を、守る。」**

## 役割 (Role)

社内 IT 環境とサイバーセキュリティを担う守りの部門。
攻撃から会社を護りつつ、社員には最高の生産性ツールを提供する。
"ゼロトラスト" を前提とした、現代的セキュアな環境を実現する。

## 主な責務

| # | 責務                       | 詳細                                            |
| - | -------------------------- | ----------------------------------------------- |
| 1 | 情報セキュリティ           | SOC 運用、脆弱性対応、ペネトレーションテスト     |
| 2 | コーポレート IT            | デバイス管理 (MDM)、SaaS 管理 (SaaS Ops)         |
| 3 | アクセス管理 (IAM)         | SSO、MFA、ゼロトラストネットワーク               |
| 4 | ITヘルプデスク             | 社員サポート、技術問い合わせ対応                 |
| 5 | インシデント対応           | CSIRT、フォレンジック、復旧                       |
| 6 | プロダクトセキュリティ     | DevSecOps、SAST/DAST、バグバウンティ              |
| 7 | コンプライアンス認証       | ISO27001、SOC2、ISMAP                            |

## 構成チーム

```
12_IT_Security/
├── InfoSec/            情報セキュリティ・SOC
├── ProductSecurity/    プロダクトセキュリティ
├── CorporateIT/        コーポレート IT
├── IAM/                認証・アクセス管理
├── Helpdesk/           ITヘルプデスク
├── CSIRT/              インシデント対応
└── Compliance/         セキュリティ認証
```

## 統括役員
- **CSO** (Chief Security Officer)

## 成功指標 (KPI)

| 指標                          | 目標値              |
| ----------------------------- | ------------------- |
| 重大インシデント数            | ゼロ                |
| 脆弱性 (Critical) MTTR        | < 24時間            |
| フィッシング訓練クリック率    | < 3%                |
| ヘルプデスク CSAT             | > 4.5 / 5.0         |
| セキュリティ認証維持          | ISO27001/SOC2 継続  |

## 行動規範

- **セキュリティは全員の仕事、専門家は触媒。**
- **侵入されないではなく、侵入されても被害ゼロ。**
- **生産性とセキュリティはトレードオフではない。**
