# V-Corp Pulse — Business Plan v0.1

> "経営の OS を、AI に置き換える。"

## 1. Executive Summary

V-Corp Pulse は、中小〜中堅企業の経営者が「毎朝3分」で
最も重要な意思決定にたどり着けるよう設計された AI 経営判断 SaaS。

- **市場**: 日本の中小企業 357万社・経営者は1日平均 **3.2時間** を「情報収集と整理」に費やしている (中小企業庁 2024)。
- **ペイン**: KPI ダッシュボードはあるが、何を見て何を決めるかは経営者が考える必要がある。
- **解決**: AI が KPI を読み解き、毎朝「今日決めるべき3つのこと」を提示する。
- **ビジネスモデル**: SaaS サブスクリプション (¥2,980〜¥198,000/月) + 高額一括 (Lifetime ¥298,000, Strategy Sprint ¥498,000)。

## 2. Problem

| ペルソナ | 現状の課題 |
| --- | --- |
| 1人会社の社長 | 数字を見るのが後回し → 月末の請求書で焦る |
| スタートアップCEO | 投資家・チーム・顧客の3方向に説明する素材作りに毎週6時間 |
| 中堅企業の事業部長 | 部門KPIが見えるが、AIで意味を解釈する仕組みがない |
| 上場企業の経営企画 | 社内データが分散、IR/取締役会向け資料に毎月20時間 |

## 3. Solution

V-Corp Pulse は次の "5本の矢" で経営判断を加速する:

1. **AI Daily Briefing** — 毎朝7:00、3行で今日の論点
2. **On-demand Strategy Reports** — チャットで指示 → 15分で20Pレポート
3. **KPI Dashboard + AI 異常検知** — 数字の "なぜ" を AI が説明
4. **Board Pack Generator** — 取締役会用エグゼ資料を自動生成
5. **Slack/Teams 連携** — 業務チャットからAIに直接質問
