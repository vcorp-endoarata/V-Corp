# 05. Marketing — マーケティング部門

## 一言ミッション
**「ブランドは、最大の資産。」**

## 役割 (Role)

V-Corp の存在を世界に知らしめ、愛されるブランドを構築する部門。
データドリブンな獲得と、感情に響くストーリーテリングを両立し、
持続可能な成長エンジンを動かす。

## 主な責務

| # | 責務                     | 詳細                                                |
| - | ------------------------ | --------------------------------------------------- |
| 1 | ブランディング           | ブランド戦略・トーン&マナー                          |
| 2 | デマンドジェネレーション | リード創出 (パフォーマンス、コンテンツ、SEO)         |
| 3 | コンテンツマーケティング | ブログ、動画、ホワイトペーパー、ニュースレター       |
| 4 | PR / コミュニケーション  | プレス対応、メディアリレーション                     |
| 5 | イベント                 | カンファレンス、ウェビナー、コミュニティ             |
| 6 | プロダクトマーケティング | GTM、ポジショニング、メッセージング                   |

## 統括役員
- **CMO** (Chief Marketing Officer)
