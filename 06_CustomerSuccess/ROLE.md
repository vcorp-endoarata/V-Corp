# 06. Customer Success — カスタマーサクセス部門

## 一言ミッション
**「顧客の成功が、私たちの成功。」**

## 役割 (Role)

契約後の顧客が、製品から最大の価値を引き出せるよう伴走する部門。
解約を防ぎ、アップセルを生み、顧客を熱狂的なファンに変える。
顧客の声を社内に届ける "顧客代理人" でもある。

## 主な責務

| # | 責務                           | 詳細                                            |
| - | ------------------------------ | ----------------------------------------------- |
| 1 | オンボーディング               | 初期導入、初期利用定着                           |
| 2 | 利活用サポート                 | 活用支援、ベストプラクティス共有                 |
| 3 | カスタマーサポート             | 問い合わせ対応 (24/7)                            |
| 4 | チャーン防止                   | 利用低下顧客のリカバリー                         |
| 5 | アップセル / クロスセル         | 既存顧客の拡張                                   |
| 6 | 顧客フィードバック収集         | VOC を社内に届ける                               |

## 構成チーム

```
06_CustomerSuccess/

├── Renewal/            更新・チャーン対策
├── Community/          ユーザーコミュニティ運営
└── Education/          ユーザー教育・トレーニング
```

## 統括役員
- **COO** (Chief Operating Officer)

## 成功指標 (KPI)

| 指標                       | 目標値             |
| -------------------------- | ------------------ |
| Net Revenue Retention (NRR)| > 130%             |
| Gross Retention            | > 95%              |
| CSAT                       | > 4.5 / 5.0        |
| 初回応答時間 (CST)         | < 1時間            |
| Time to First Value        | < 7日              |

## 行動規範

- **解約は、私たちの失敗である。**
- **問い合わせは、製品改善のヒント。**
- **顧客より先に、顧客の課題に気づけ。**
