# 11. R&D — 研究開発部門

## 一言ミッション
**「10年後の事業を、今日作る。」**

## 役割 (Role)

V-Corp の未来の競争力を生み出す研究組織。
基礎研究、応用研究、Moonshot プロジェクトを通じて、
誰もまだ気づいていない未来の価値を発掘する。

## 主な責務

| # | 責務                          | 詳細                                            |
| - | ----------------------------- | ----------------------------------------------- |
| 1 | AI / ML 研究                  | 基礎モデル、応用研究、論文発表                   |
| 2 | 新規事業インキュベーション    | 0→1 プロジェクト                                  |
| 3 | プロトタイピング              | コンセプト検証、ラピッドプロトタイプ             |
| 4 | 学術連携                      | 大学・研究機関との共同研究                       |
| 5 | 特許戦略                      | 防衛特許・攻撃特許のポートフォリオ構築           |
| 6 | テックスカウティング          | 新技術・スタートアップの調査                     |
| 7 | サイエンスコミュニケーション  | 論文、技術ブログ、カンファレンス登壇             |

## 構成チーム

```
11_RnD/
├── AI_Research/        AI / ML 基礎研究
├── Applied/            応用研究 (CV / NLP / Robotics)
├── Labs/               プロトタイピング・ラボ
├── Incubation/         新規事業インキュベーション
├── Academic/           学術連携・大学コラボ
└── Scouting/           テクノロジースカウティング
```

## 統括役員
- **CTO** (Chief Technology Officer)

## 成功指標 (KPI)

| 指標                           | 目標値          |
| ------------------------------ | --------------- |
| 主要学会論文採択数             | 年 30本以上     |
| 特許出願数                     | 年 50件以上     |
| 研究 → 製品化率                 | > 20%           |
| Moonshot プロジェクト          | 常時 5本以上    |
| R&D 投資 / 売上比               | 15% 以上        |

## 行動規範

- **失敗の数だけ、未来が近づく。**
- **論文より、世界を変えるプロダクト。**
- **常識を疑え。されど基礎を尊べ。**
